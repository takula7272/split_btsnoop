1. put the tool and btsnoop file under the same path
2. cmd: oerl split_btsnoop.pl
3. > Input btsnoop file name:
   input the btsnoop file name (no need to inlude file extension)
4. > Input byte size(MB):
   input the target file unit size, ex: if the btsnoop total size is 900MB and input 10, 90 files would be generated
5. console will show the progress
6. Output file name is the <original file name> + number + "cfa"
